# BridgeAI - Sovereign Context Protocol

BridgeAI is a professional context-relay system that connects workflows across different AI platforms like ChatGPT, Claude, and Gemini. It mitigates context loss, accelerates multi-LLM development velocities, and ensures sensitive logical constraints are seamlessly synchronized.

## Core Platform Workflow

1. **Orientation**: Access the dashboard and install the companion extension to enable live extraction. 
2. **Context Capture (Bridging)**: During an intensive problem-solving session with an LLM (e.g., ChatGPT), capture the raw chat via the extension or manually via the dashboard's manual extractor.
3. **Smart Summarization**: BridgeAI captures the raw data tokens, extracting the core project intelligence (goals, technical stack, current bugs, constraints).
4. **Secure Vaulting**: The encrypted intelligence bundle (the "Bridge") is saved permanently to the Sovereign Hub Vault under your project folder.
5. **Universal Forging (Relay)**: To continue the session elsewhere, click "Forge Universal Bridge". The platform compiles a highly optimized Prompt Core and automatically dispatches it to a selected destination (e.g., Claude, Gemini, Perplexity) for continuous generation.

## Technical Architecture

- **Frontend**: React (Vite) integrating Framer Motion for high-fidelity animations and dynamic UI feedback. 
- **Backend API (`server/`)**: Express-driven Node.js handling cross-origin requests, authentication routing, and subscription states.
- **Database Architecture**: PostgreSQL housing user profiles, invoices, subscription tiers, and local telemetry.
