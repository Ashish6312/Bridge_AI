# BridgeAI - Sovereign Context Protocol 🚀

**Sovereign Intelligence Orchestration for Multi-LLM Workflows.**

BridgeAI is a professional-grade context-relay system designed to bridge the intelligence gap between disparate AI platforms (Gemini, Claude, ChatGPT, Perplexity). It ensures seamless context continuity, accelerates development velocity, and preserves logical constraints across sessions.

---

## 💎 Elite Features

### 🏛️ Sovereign Vault
A centralized, high-fidelity dashboard for managing extracted intelligence. Features glassmorphic design, real-time telemetry, and hierarchical project organization.

### 🪄 Intelligence Forge
Automated prompt engineering core that re-distills raw chat logs into highly optimized "Intelligence Bundles" ready for consumption by other LLMs.

### 📡 Analyst Module (Extension)
A standalone browser extension that enables live, one-click extraction from any compatible AI interface, relaying context directly to your Sovereign Vault.

### ⚡ Universal Dispatch
Protocol-based routing that allows you to "Forge" and "Dispatch" intelligence to multiple destinations simultaneously via clipboard, email, or direct hub links.

---

## 🛠️ Technical Architecture

### **Frontend Infrastructure**
- **Framework**: React.js with Vite for lightning-fast HMR.
- **Motion Engine**: `framer-motion` for professional-grade micro-animations and transitions.
- **Iconography**: Lucide-React (Curated professional set).
- **Styling**: Vanilla CSS with custom design tokens for glassmorphism and premium aesthetics.

### **Backend & Storage**
- **Runtime**: Node.js (Express) with robust cross-origin intelligence handling.
- **Distillation**: Integrated AI summarization logic (Pollinations/OpenAI compatible).
- **Security**: Sovereign identity lock and multi-layered data isolation.

### **Extension Bridge**
- **Technology**: Manifest V3 browser extension.
- **Sync Protocol**: DOM-event based relay with secure vault mirroring.

## 🚀 Quick Start

### **1. Repository Initialization**
```bash
npm install
cd server && npm install
```

### **2. Launch Local Hub**
```bash
# In root
npm run dev

# In server directory
node index.js
```

### **3. Operational Deployment**
1. Access the Dashboard at `localhost:5173`.
2. Install the **Analyst Module** via the 'Extension' tab.
3. Paste context in **Manual Bridge** or use the extension to capture live logs.
4. Forge and Dispatch intelligence to your target LLM.
